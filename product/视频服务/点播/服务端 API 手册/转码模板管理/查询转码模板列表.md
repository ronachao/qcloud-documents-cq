## 接口名称
QueryTranscodeTemplateList

## 功能说明
1. 查询转码模板列表。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| COMMON_PARAMS | 是 |  | 参见[公共参数](/document/api/213/6976) |

### 请求示例
<pre>
http://vod.api.qcloud.com/v2/index.php?Action=QueryTranscodeTemplateList
&COMMON_PARAMS
</pre>

## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码, 0: 成功, 其他值: 失败 |
| codeDesc | String | 详细信息 |
| data | Array | 转码模板信息列表 |
| data.definition | Integer | 转码模板 ID |
| data.create_time | Integer | 模板创建时间（Unix 时间戳） |
| data.update_time | Integer | 模板信息最近更新时间（Unix 时间戳） |


### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见[公共错误码](/document/product/266/7783)  |
| 1000 | 无效参数  |
| 10702 | 内部错误  |
