主从版 Redis 是云平台基于开源的 Redis 打造的一款高可用、高可靠的 Redis 服务。通过主从热备自动容灾提供高可用性，通过多份数据拷贝保证高可靠性。主从版 Redis 支持开源 V2.8 版本协议，容量支持范围为 1~60G。
