1.登录 云平台 管理控制台 并通过以下网址打开  控制台：[https://console.cloud.tencent.com/tdsql](https://console.cloud.tencent.com/tdsql)

2.在 管控控制台 的右上角，点击新建实例；或直接打开售卖页面[https://buy.cloud.tencent.com/tdsql](https://buy.cloud.tencent.com/tdsql)


3.在购买页面上，根据需求指定数据库实例信息，点击【立即购买】；
>购买时，注意以下设置：
>
>- 计费模式：按量计费，即无需预先支付费用即可使用，使用完成后再结算。
>- 地域：实例实际部署的地域；建议您于需要对接的云服务器保持一致，以便延迟最低。
>- 可用区：在同一地域内电力和网络互相独立的物理数据中心；建议您于需要对接的云服务器保持一致，以便延迟最低。
>- 网络类型：实例所处的网络，一旦网络选定就不能轻易改变；建议您于需要对接的云服务器保持一致，以便延迟最低。
>- 实例版本：请根据需要选择您期望的[实例版本](https://cloud.tencent.com/doc/product/237/6918)；通常来讲，备机越多，可用性越高。
>- 数据库版本：MySQL/MariaDB(TDSQL)的数据库内核版本，由于各个版本之间存在功能差异，详见[MariaDB官方版说明](https://mariadb.org/)，[MySQL官方版说明](https://www.mysql.com/)
>- 实例规格：实例规格代表不同的性能水平和价格基数,详见[产品价格]（https://cloud.tencent.com/document/product/237/2034）。
>- 硬盘：硬盘默认采用SSD盘（本地盘），更多硬盘种类敬请期待。
>- 购买数量：指一次性可购买的实例个数，*为避免误操作，我们设置了一次购买上限，如果您期望购买更多台数，请多次购买*。



4. 购买后将弹出二次确认弹窗，您需要二次确认后，系统才能分配刚购买的实例。
