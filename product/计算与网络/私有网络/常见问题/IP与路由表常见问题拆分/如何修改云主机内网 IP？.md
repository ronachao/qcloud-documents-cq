云主机主网卡的主内网 IP 支持修改，辅助网卡的主内网 IP 不支持修改，操作步骤如下：
1. 进入 [云服务器控制台](http://console.tcecqpoc.fsphere.cn/cvm/)，点击左导航栏的云主机，进入云主机列表页。
2. 点击云主机 ID，进入云主机详情页，点击上方 tab：弹性网卡。
3. 点击修改主 IP。
4. 填入新的 IP，并保存即可。
