配置说明：
1）后端对接 http 时，需要选择您的后端类型为 http 或 https。

2）输入后端地址，以 `http://` 或 `http://` 开头，不包括后面的路径，例如 `http://api.myservice.com` 或 `http://108.160.162.30`。

3）输入后端路径以 / 开头，如 `/path` 或 `/path/{petid}`。

4）选择请求方法，前后端选择的请求方法可不一致。

5）设置后端超时时间。

6）设置映射前端的后端参数。

7）单击【完成】。
