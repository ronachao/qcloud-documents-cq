## 1. 接口描述

本接口 (SetTopicAttributes) 用于在CKafka实例下修改主题属性。

接口请求域名：<font style="color:red">ckafka.api.qcloud.com</font>

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数，其它参数见[公共请求参数](/doc/api/431/5883)页面。

| 参数名称 | 必选 | 类型 | 描述 |
| --- | --- | --- | --- |
| instanceId | 是 | String | 实例id |
| topicName | 是 | String | 主题名称 |
| enableWhiteList | 否 | int | ip白名单开关, 1:打开  0:关闭 |


## 3. 示例

输入：

```
 http://domain/v2/index.php?Action=SetTopicAttributes&instanceId=ckafka-xxxxxx&topicName=xxxxx&<公共请求参数>
```

输出：

```
  {
      "code" : 0,
	"codeDesc":"Success"
      "message" : "ok",
  }

```
